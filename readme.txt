This plugin communicates with your WPSupervisor Admin Panel.

== Description ==

[WPSupervisor](http://www.profitmarketer.com/services/wp-supervisor/) allows users to manage unlimited number of WordPress sites from their account.

Main features:

*   One-click updates for WordPress, plugins and themes across all your sites
*   Instant backup and restore your entire site or just the database
*   One-click access to all WP admin panels
*   Bulk Manage plugins & themes: Activate & Deactive multiple plugins & themes on multiple sites simultaneously
*   Bulk Install plugins & themes in multiple sites at once
*   and more..

Visit us at [WPSupervisor](http://www.profitmarketer.com/services/wp-supervisor/).

Check out the [WPSupervisor Overview Video](http://www.youtube.com/watch?v=5HFY4iYifbE).

Credits: [InfiniteWP](http://infinitewp.com/) for InfiniteWP Client and [Vladimir Prelovac](http://prelovac.com/vladimir) for his worker plugin on which the client plugin is being developed.


== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder
2. Go to the Plugins page and activate WPSupervisor Client
3. If you have not activated the WPSupervisor account, visit [WPSupervisor](http://www.profitmarketer.com/services/wp-supervisor/).
4. Add your WordPress site to the WPSupervisor Admin Panel and start using it.
